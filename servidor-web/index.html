Servidor web activo

Proyecto Final - Integracion Continua

Autor: Johan Sebastian Rodriguez.

Este contenedor ejecuta un servidor Nginx y permite validar la comunicacion con un segundo contenedor de pruebas.

La comunicacion interna se realiza mediante la red Docker Compose y el nombre del servicio servidor-web.

Resultado esperado: el contenedor cliente-pruebas debe conectarse a este servidor mediante ping y curl.
