Esta carpeta debe contener únicamente capturas reales tomadas durante la ejecución y la sustentación.

## Criterio de aceptación

Cada captura debe permitir identificar:

1. La herramienta utilizada.
2. La acción ejecutada.
3. El resultado obtenido.

No incluir capturas de resultados simulados. Para cada figura insertada en el documento final se debe agregar título, explicación breve y la fuente: **Elaboración propia**.

## CodeShip

La evidencia `12-codeship-build.png` solo debe incluirse cuando exista un build real. Si no se dispone de acceso operativo, documentar la limitación en el informe y no crear una captura simulada.
