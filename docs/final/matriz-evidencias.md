## Capturas obligatorias

1. Repositorio GitHub y `README.md`.
2. `docker-compose.yml` y `docker compose ps`.
3. Salida del script de comunicación entre contenedores.
4. Jenkins: pantalla inicial, plugins y usuario configurado.
5. Jenkins: tarea Pipeline y consola de ejecución.
6. `.travis.yml` en GitHub.
7. Travis CI: repositorio activado y resultado del build.
8. Historial de commits y, si existe, pull request.
9. Evidencia de CodeShip solo si existe acceso y build real.

## Criterio de aceptación

Una evidencia se considera válida cuando permite identificar la herramienta, la acción ejecutada y el resultado obtenido. Las capturas deben llevar número de figura, título, explicación y fuente: **Elaboración propia**.
