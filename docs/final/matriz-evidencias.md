# Matriz de evidencias para la entrega final

| Requisito de sustentacion | Evidencia que debe mostrarse | Estado actual | Responsable |
|---|---|---|---|
| Repositorio y control de cambios | GitHub: README, commits, archivos de configuracion | Preparado | Integrante 1 |
| Contenedores comunicados | `docker compose ps` y `validar_comunicacion.sh` | Requiere captura local | Integrante 2 |
| Jenkins como gestor de operaciones | Dashboard, plugins, Pipeline y consola | Requiere ejecucion local | Integrante 3 |
| Requisitos de Jenkins | Tabla de hardware, software, dependencias y puertos | Documentado | Integrante 3 |
| Integracion Travis CI - GitHub | `.travis.yml`, activacion y build | Configuracion preparada; requiere activacion y captura | Integrante 4 |
| Historial de cambios | GitHub commits, rama o pull request | Preparado | Integrante 4 |
| Responsabilidades y opiniones | Tabla de roles y breve conclusion de cada integrante | Pendiente de completar con nombres reales | Todo el equipo |
| CodeShip | Acceso activo, configuracion y build o nota formal de limitacion | Pendiente de validacion externa | Equipo |
| Documento de soporte | Portada, figuras explicadas, APA y referencias | Pendiente de ensamblaje final | Integrante 5 |
| Video final | Sustentacion de 5 a 6 minutos con participacion del equipo | Pendiente de grabacion | Todo el equipo |

## Capturas obligatorias

1. Repositorio GitHub y `README.md`.
2. `docker-compose.yml` y `docker compose ps`.
3. Salida del script de comunicacion entre contenedores.
4. Jenkins: pantalla inicial, plugins y usuario configurado.
5. Jenkins: tarea Pipeline y consola de ejecucion.
6. `.travis.yml` en GitHub.
7. Travis CI: repositorio activado y resultado del build.
8. Historial de commits y, si existe, pull request.
9. Evidencia de CodeShip solo si existe acceso y build real.

## Criterio de aceptacion

Una evidencia se considera valida cuando permite identificar la herramienta, la accion realizada y el resultado obtenido. Las capturas deben llevar numero de figura, titulo, explicacion y fuente: elaboracion propia.
