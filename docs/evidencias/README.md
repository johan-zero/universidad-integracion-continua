# Evidencias de ejecución

En esta carpeta se deben almacenar las capturas de pantalla que soportan la Entrega 1.

## Capturas sugeridas

1. Repositorio publicado en GitHub.
2. Estructura de archivos del repositorio.
3. Ejecución de `docker compose build`.
4. Ejecución de `docker compose up -d`.
5. Resultado de `docker compose ps`.
6. Resultado de `docker compose exec cliente-pruebas validar_comunicacion.sh`.
7. Vista del navegador en `http://localhost:8080`.

## Recomendación de nombres

Usar nombres claros, por ejemplo:

```text
01-repositorio-github.png
02-estructura-repositorio.png
03-docker-build.png
04-docker-up.png
05-docker-ps.png
06-validacion-comunicacion.png
07-navegador-localhost.png
```
