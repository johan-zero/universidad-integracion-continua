# Entrega 2 - Implementacion de Jenkins

Esta carpeta contiene la documentacion tecnica de la segunda entrega del modulo Integracion Continua.

## Alcance

La Entrega 2 complementa la Entrega 1. Se conserva el repositorio GitHub y el ambiente Docker previamente creado, y se incorpora Jenkins como gestor de operaciones para automatizar una validacion basica del proyecto.

## Evidencias esperadas

1. Instalacion o despliegue de Jenkins.
2. Acceso a Jenkins desde navegador.
3. Instalacion de plugins recomendados.
4. Creacion de usuario administrador.
5. Creacion de una nueva tarea o pipeline.
6. Conexion del pipeline con el repositorio GitHub.
7. Ejecucion del Jenkinsfile.
8. Validacion de construccion Docker y comunicacion entre contenedores.

## Archivos relacionados

- `docker-compose.jenkins.yml`: define el servicio Jenkins.
- `Jenkinsfile`: define las etapas del pipeline.
- `docs/entrega-2/guia-instalacion-jenkins.md`: guia de instalacion y configuracion.
- `docs/entrega-2/evidencias/README.md`: guia para capturas.
